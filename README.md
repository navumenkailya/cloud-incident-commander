# Cloud Incident Commander (AIOps Platform)

Проект представляет собой отказоустойчивую мультиоблачную платформу автоматического сбора логов, метрик и интеллектуального анализа инцидентов (AIOps) на базе локальных LLM-моделей.

## 🏗️ Архитектура системы

Система эмулирует работу двух независимых облачных контуров (AWS и Yandex Cloud), развернутых в изолированных Docker-сетях.

  +------------------+         +------------------+
  |    Cloud AWS     |         |   Cloud Yandex   |

           +--------------+-------------+
                          | (JSON Logs & Metrics)
                          ▼
                 [ Central Broker ]
                          |
